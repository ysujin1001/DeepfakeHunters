데이터베이스 구조 (SQLite)
🗄️

👤 users (사용자 정보)

  • user_id (PK) - 사용자 고유 ID
  • email - 이메일 (UNIQUE)
  • password_hash - 암호화된 비밀번호
  • created_at - 가입일시

📊 analysis_results (분석 결과)

  • result_id (PK) - 결과 고유 ID
  • user_id (FK) - 사용자 ID
  • image_path - 원본 이미지 경로
  • analysis_type - 탐지/복원 구분
  • result_data - JSON 형식 결과
  • confidence_score - 신뢰도 점수
  • created_at - 분석일시

🏷️ metadata (메타데이터)

  • metadata_id (PK) - 메타데이터 ID
  • result_id (FK) - 분석 결과 ID
  • face_features - 얼굴 특징 벡터
  • age_estimate - 추정 연령
  • gender - 성별
  • emotion - 감정 상태
